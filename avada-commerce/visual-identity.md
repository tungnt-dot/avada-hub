### Gradient
Avada uses **Freeform Gradient** as the primary visual method. It combines brand colors in a customized, fluid gradient style that reflects creativity, modernity, and flexibility.

## Typography

### Primary Font: Poppins
- Geometric sans-serif
- Used for both headings and body text
- Min size: 7pt (print), 9pt (digital)
- Limitation: does not support Vietnamese

### Supporting Fonts
- **Bai Jamjuree** — Square sans-serif, good for headings, supports Vietnamese
- **Averta** — Used for body text, supports Vietnamese
- Max 2 fonts per design

## Logo
- **Brandmark**: Spotlight-inspired icon with freeform gradient
- **Wordmark**: "avada" in lowercase with custom font
- **Signature**: Brandmark + Wordmark combined (preferred usage)
- Always maintain safe zones and minimum sizes
- Min digital width: Brandmark 30px, Wordmark 60px, Signature 62px

## Visual System (3 patterns)
All derived from the Spotlight concept:

1. **Sparkling** — Light rays on sky, ratio 1:1, height matches design height
2. **Combination** — Flexible intersection of shapes
3. **Attention** — Two shapes with movable intersection points

Preferred backgrounds: White or Navy Blue.

## Icon System
Custom icons using brand colors (Navy Blue, Cerulean, Bittersweet, Cherokee) with Spotlight concept. Optimized for digital at 50px+ width.

Available icons: Shield, Clock, Chat, Raise Sales, Dollars, SEO, Loading, Internet, Warning, Setting, Speedometer, Rocket, Analytic, Email, SMS, Workflow, Notifications, Segment, Customer, Invoice, Size Chart, FAQs, Cookie.
